Tabber

WELCOME

Tabber is a Google Extension made to allow users to organize and reformat their windows and tabs without using the mouse or trackpad.

BEFOREYOU BEGIN

This tool offers up to 22 unique keyboard shortcuts that, when used alongside existing Chrome shortcuts, allows users to complete a variety of tasks without every needing to take their fingers off of the keyboard.

There are 3 instant commands that can be performed at any time - moving the tabs are actually existing Chrome Commands that I listed since many users are unaware of it.

For all other commands they can ONLY BE USED when the extension popup is open, which is why I highly recommend choosing a convenient shortcut for opening the popup itself. Also, to enable all commands, please allow this extension to be run in incognito mode - this can be done by going to "Details" when on the chrome://extensions/ page. (It doesn't store any data) Also, before you begin, adjust your screen to the normal screen size and open the popup. Then press P, as this will allow the extension to register your screen size.

RECOMMENDED EXTENSION OPENING COMMAND: Cmd Shift A

ALL SHORTCUTS

INSTANT COMMANDS

Open extension: Customizable

Undo a split tab/window: Cmd Shift Up

Move tab left: Cmd Shift Left

Move tab right: Cmd Shift Right

Go to left of current tab: Cmd Alt Left

Go to right of current tab: Cmd Alt Right

WINDOW RESIZE

Tab to new new window & split screen: D

Normal-sized screen: Z

Fullscreen Mode: X

Left half of screen: Left

Right half of screen: Right

Top half of screen: Up

Bottom half of screen: Down

HIGHLIGHT & PIN

Highlight left of current tab: Alt Left

Highlight right of current tab: Alt Right

Unhighlight right of current tab: Alt Up

Unhighlight left of current tab: Alt Down

Pin tab: M

Unpin tab: N

FLIP & GROUP

Group all incognito windows together: W

Group all regular windows together: Q

Flip through all open windows: F

SORTING TABS

Sort all tabs by URL: A

Sort all tabs by titles: S

*Windows equivelant of "Cmd" is the Windows key
